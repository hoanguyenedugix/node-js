# Node.js Tool Guide

## Overview
This tool provides a set of Node.js utilities to help developers streamline their workflow. This guide will help beginners get started with using these tools effectively.

## Installation

**Prerequisites:**

*   Node.js (v20 or higher recommended)
*   npm

**Steps:**

1.  Install dependencies.
    npm install

2. Edit file env
   **Config folder HTML**
   LESSON_DIR=./html_lesson_plan (example)


   **Config enviroment to push lession**
    ENV=prod

3. Start
    npm start
